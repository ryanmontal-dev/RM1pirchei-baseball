# Shkoyach Sluggers

A mobile-first Phaser 3 Home Run Derby game with backyard baseball-inspired art direction, team and character selection, practice and Pirchei Championship modes, PWA support, and local save progress.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Controls

- Tap during the pitch for a contact swing.
- Swipe down during the pitch for a power swing.

## Features

- Mobile-first 390x844 portrait game canvas with responsive scaling.
- Backyard-inspired hand-drawn UI made with Phaser primitives.
- Team selection and character selection.
- Practice mode with 12 swings.
- Pirchei Championship mode with 10 swings, a 7-home-run target, and bonus swings.
- Shkoyach fireworks after every home run.
- Home run distance tracking and career records.
- Streak scoring, distance bonuses, and swing history dots.
- Persistent progress via localStorage.
- PWA manifest, service worker, and installable app icons.
